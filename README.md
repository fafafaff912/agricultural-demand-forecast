#  Прогноз спроса на сельхозпродукцию

## Описание задачи

Прогнозирование объёмов производства/продаж сельскохозяйственной продукции
(пшеница, молоко, кукуруза и др.) на основе временных рядов.
Задача: предсказание значений на 12 месяцев вперёд с учётом сезонности и трендов.

## Цель и мотивация

- Планирование закупок и логистики в агросекторе
- Оптимизация складских запасов
- Снижение потерь от перепроизводства / дефицита
- Поддержка принятия решений для трейдеров и фермеров

## Данные

**FAOSTAT** — открытые данные ФАО ООН по производству, торговле
и ценам на сельхозпродукцию по странам и годам/месяцам.

- Источник: [FAOSTAT](https://www.fao.org/faostat/en/#data)
- Дополнительно: [USDA Market News](https://marketnews.usda.gov/)
- Скрипт автоматически скачивает данные через API FAOSTAT
- Признаки: объём производства, цены, экспорт/импорт, погода (опционально)

## Архитектура моделей

### Сравниваемые модели:

| Модель | Тип | Описание |
|--------|-----|----------|
| **ARIMA** | Baseline | Классическая модель временных рядов |
| **Prophet** | Baseline | Facebook Prophet — аддитивная декомпозиция |
| **N-BEATS** | Neural | Neural Basis Expansion для временных рядов |
| **Temporal Fusion Transformer (TFT)** | Neural | Attention-based модель с интерпретируемостью |

### Обоснование выбора основной модели — N-BEATS:

- Не требует ручного feature engineering
- Автоматически выделяет тренд и сезонность (interpretable архитектура)
- Лёгкая и быстрая в обучении по сравнению с TFT
- State-of-the-art на M4 Competition

## Метрики качества

| Модель | MAPE ↓ | SMAPE ↓ | MAE ↓ | RMSE ↓ |
|--------|--------|---------|-------|--------|
| ARIMA | ~12-18% | ~11-16% | - | - |
| Prophet | ~10-15% | ~9-14% | - | - |
| N-BEATS | ~6-10% | ~6-9% | - | - |
| TFT | ~5-9% | ~5-8% | - | - |

*Конкретные значения зависят от выбранного продукта и региона.*

Графики прогнозов и сравнения сохраняются в `results/`.
